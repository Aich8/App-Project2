# Cash Money Organizer Website

This folder contains the Next.js website application.

## Run Locally

From the repository root:

```powershell
cd website
npm run dev
```

Then open `http://localhost:3000`.

## App Files

- `app/page.tsx`: main website page.
- `app/layout.tsx`: root layout.
- `app/globals.css`: global styles.
- `public/`: static assets.

Project specs and plans live outside this app folder in `../docs/`.
